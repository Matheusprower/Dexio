# Dexio

Objetivo: Ser uma versão de PokéDex funcional e usual para mim próprio, contendo funções que ativamente me ajudem a zerar o jogo e rastrear o meu objetivo de zerar com Pokémon diferente. O programa deve ser feito inteiramente por mim.

Precisa ter:
* Atualização automática - Sem precisar adicionar Pokémon e suas informações conforme novos jogos vão sendo lançados.
* Multi-Usuários - Caso eu queira ter mais de uma visão de usuário eu possa, tendo uma forma fácil e simples de trocar de usuário.
* Exibir informações dos Pokémon - Pelo menos o nome, a imagem, os tipos, onde ele é capturado e os golpes que ele aprende.
* Track de PokéDex - Consiga marcar Pokémon capturados e usados.
* Track de Jornadas - Consiga marcar as jornadas que o usuário já fez, tendo um registro e uma forma visual de aprecisar, como um hall da fama por exemplo.
* Várias PokeDex - A informação exibida deve ser do jogo selecionado.

Técnologias:
* Backend - Java
* Frontend - React
* Hospedagem em docker e com banco de dados.
